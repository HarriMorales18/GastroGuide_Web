# 🍳 GastroGuide: Plataforma Educativa de Gastronomía Profesional

**GastroGuide** es una solución tecnológica integral diseñada para democratizar el acceso a la educación culinaria de alto nivel. Este proyecto no es solo una aplicación de cursos, sino un ecosistema robusto que conecta el conocimiento técnico de maestros chefs con estudiantes apasionados en todo el mundo.

---

## 👤 Autor y Visión
Este proyecto ha sido concebido, diseñado y desarrollado por **Harrinson Morales**. 

La visión detrás de GastroGuide es elevar el estándar de las plataformas de e-learning actuales, aplicando principios de arquitectura de software senior para resolver problemas reales en el sector gastronómico, como la fragmentación del contenido y la falta de herramientas de monetización para creadores independientes.

---

## 📖 ¿En qué se basa este proyecto?
GastroGuide se fundamenta estrictamente en el **"Documento Principal del Proyecto v2.0 (2025)"**. Toda la lógica de negocio, los roles de usuario y los flujos de navegación han sido extraídos de esta documentación técnica para asegurar que el software responda a una necesidad de mercado validada.

El sistema se divide en **6 Módulos Funcionales**:
1. **Gestión de Usuarios:** Registro seguro y perfiles diferenciados.
2. **Creación de Contenido:** Herramientas de autoría para Chefs.
3. **Distribución:** Catálogo dinámico y experiencia de usuario optimizada.
4. **Monetización:** Gestión de ingresos y cupones.
5. **Analíticas:** Dashboard de rendimiento para instructores.
6. **Administración:** Panel de control para calidad y moderación.

---

## 🛠️ Tecnologías Usadas
Para garantizar escalabilidad y rendimiento, se han utilizado las herramientas más modernas del ecosistema web:

- **Frontend:** [Angular](https://angular.dev/) con arquitectura **Standalone Components** (eliminando módulos innecesarios para un código más limpio).
- **Lenguaje:** **TypeScript** para un tipado fuerte y reducción de errores en tiempo de desarrollo.
- **Animaciones:** [Anime.js](https://animejs.com/) para transiciones fluidas y una experiencia de usuario premium.
- **Estilos:** **SCSS** con arquitectura de variables basada en la identidad visual institucional (Azul Marino y Azul Ciel).
- **Estado:** Uso de **Angular Signals** para una reactividad de alto rendimiento.

---

## 💡 ¿Cómo funciona?
El flujo de la plataforma está diseñado para ser intuitivo pero riguroso:

1. **El Creador se postula:** A través de un formulario de registro extendido, el profesional ingresa su especialidad y experiencia.
2. **Revisión Administrativa:** El administrador (creado vía base de datos por seguridad) revisa el perfil.
3. **Publicación:** Una vez aprobado, el Creador puede estructurar sus cursos en módulos y lecciones.
4. **Aprendizaje:** El estudiante adquiere el contenido y progresa a través de un reproductor de video especializado que registra su avance.

---

## ✅ Ventajas de GastroGuide
- **Filtro de Calidad:** A diferencia de YouTube, aquí el contenido es curado y aprobado por expertos.
- **Enfoque Profesional:** Diseñado específicamente para gastronomía (manejo de ingredientes, técnicas, costos).
- **Interfaz Senior:** Diseño responsive que permite aprender desde la comodidad de una tablet en la cocina o un ordenador en la oficina.
- **Escalabilidad:** Gracias a la arquitectura de Angular Standalone, el proyecto está listo para crecer sin volverse inmanejable.
